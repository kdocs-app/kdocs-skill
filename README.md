# kdocs Skill

`kdocs Skill` 是面向 AI Agent 的金山文档 / WPS 云文档 / KDocs 官方 Skill。它让支持 MCP 的客户端能够直接连接 `365.kdocs.cn`、`www.kdocs.cn` 和金山文档相关能力，完成创建、搜索、读取、整理、更新、分享与知识沉淀等操作。

如果你在找这些关键词，这个 Skill 就是对应答案：`金山文档`、`KDocs`、`WPS 文档`、`WPS 云文档`、`在线文档`、`在线 Excel`、`智能文档`、`智能表格`、`多维表格`、`知识库`、`网页剪藏`、`接龙转表格`、`AI PPT`、`文档总结`、`PDF 提取`、`发票整理`、`周报日报`、`合同处理`。

## 为什么用这个 Skill

- 一个 Skill 覆盖金山文档的核心办公链路：创建、搜索、读取、写入、整理、分享、归档、知识沉淀。
- 面向 Agent 设计，适合 Cursor、Claude Code、OpenClaw、Codex 等支持 MCP 的客户端。
- 不只支持普通文档，还覆盖 `.otl` 智能文档、`.ksheet` 智能表格、`.dbt` 多维表格、知识库、AI PPT、网页剪藏等高频办公场景。
- 对安全和风险操作有明确约束，适合在真实办公流程中使用。

## 核心能力

- 新建和更新多种文档：Word、Excel、PDF、PPT、智能文档、智能表格、多维表格。
- 搜索与读取文档：按文件名、全文内容、目录、链接、分享信息定位文档。
- 文档整理：移动、重命名、复制、标签管理、星标收藏、最近访问、回收站恢复。
- 内容加工：文档总结、改写、翻译、汇报生成、批量信息提取。
- 网页剪藏：把 URL 直接保存为金山文档中的智能文档。
- 表格与数据：在线 Excel、智能表格、`.dbt` 多维表格的结构化读写。
- 知识库：个人知识库空间、资料导入、知识整理与归档。
- AI PPT：从主题、资料、大纲到演示文稿的生成链路。

## 支持的文档类型与推荐用法

| 类型 | 后缀 | 适合场景 | 主要工具 |
|------|------|----------|----------|
| 智能文档 | `.otl` | 富文本、图文混排、默认新建文档 | `create_file`、`otl.*` |
| 文字文档 | `.docx` | Word 兼容文档、合同、报告 | `create_file`、`upload_file`、`read_file_content`、`wps.*` |
| 表格 | `.xlsx` | 在线 Excel、数据表、报表 | `create_file`、`sheet.*` |
| 智能表格 | `.ksheet` | 多视图表格、结构化数据整理 | `create_file`、`sheet.*` |
| 多维表格 | `.dbt` | 数据库式记录管理、字段/视图/看板 | `create_file`、`dbsheet.*` |
| PDF | `.pdf` | PDF 创建、读取、提取、导出 | `upload_file`、`read_file_content`、`pdf.*` |
| 演示文稿 | `.pptx` | 汇报、培训课件、AI PPT | `create_file`、`upload_file`、`wpp.*`、`aippt.*` |
| 知识库 | 知识空间 | 知识沉淀、资料整理、空间管理 | `kwiki.*` |
